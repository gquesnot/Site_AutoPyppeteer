Exemples

Creation of a Script that will add a new project on my own Web App

1. Enter project name and click New project
2. We are going to create 2 blocks for this project, the first one that we will call "login on AApy" for init / reuse and an other block for the Core part that we will call "create a new project on AppPy"

Select "login on AApy"

Select "login on AApy"

3. Go on action part to create the login step

First, create a go_to Action and copy the login url as value

Here you have to create 3 Actions to get logged in :

  • Input: to fill the username
  • Input: to fill the password
  • Click: on the button

add 1 action screenshot to check if you are really logged In

4. Now you can go back to the top of the page to set the block "login on AApy" as init , update and run Project.

after a few second you should see a popup with your picture and title

5. Next step is to configure the block to create a new project

select the other block

we have 3 step:

  • Go_to: projects page
  • Input: fill the name of your new project
  • Click: click on create
  • Screenshot: to check if it's done
6. On the top of the page, select "new project on AApy" as Core Block

update and run Project
